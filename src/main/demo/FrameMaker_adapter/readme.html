Running the FrameMaker adapter for the DITA Open Toolkit

This tutorial demonstrates how to generate sample documents in the DITA Open Toolkit using Ant and view the ouptut in Adobe FrameMaker.

The following tasks require the DITA Open Toolkit 1.2 or later and Adobe FrameMaker 7.0 or later. (If you are not using FrameMaker 7.2, you need to edit the value of the FrameMakerExecutable property in integrator.xml.)

Before you begin, extract the contents of the FrameMaker_adapter.zip archive to DITA-0T1.2.1\demo. You must also install the adapter plug-in and add the supplied DITA-OT application to your FrameMaker installation. For information about working with a structured applications file, refer to the Structure Application Developer's Guide that is distributed with FrameMaker.

To install the adapter plug-in

  1. Open a command window and change the path to the DITA-OT1.2.1 folder.
  2. On the command line, type ant -f integrator.xml and press Enter.

To add the DITA-OT application to your FrameMaker installation

  1. Start FrameMaker and open Open DITA-0T1.2.1\demo\FrameMaker_adapter\DITAFM_app.fm.
  2. Click the Structure View button () at the upper right corner in the document window.
  3. Select and copy the DITA-OT XML application.
  4. Open structapps.fm in your FrameMaker Structure folder (for example, C:\Program Files\Adobe\FrameMaker7.2\Structure).
  5. In the Structure View, move your cursor below the last XML application and paste the DITA-OT application. If necessary, edit the pathnames in the application.
  6. Select File>Structure Tools>Read Application Definitions.

To generate sample FrameMaker output

  1. On the command line, type ant -Dargs.input="samples\sequence.ditamap" dita2fmxml and press Enter. Output is copied to C:\DITA-OT1.2.1\out and FrameMaker is started.
  2. In the Use Structured Application dialog, select the DITA-OT application and press Continue. FrameMaker opens the output using the DITA-OT application.
