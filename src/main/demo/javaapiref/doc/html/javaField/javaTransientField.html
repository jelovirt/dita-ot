javaTransientField

The <javaTransientField> element declares within the FieldModifiersopt that the fields introduced by the declaration are marked transient fields.

Description

This element specifies that the value of the field should not be serialized. The variables can be marked transient to indicate that they are not part of the persistent state of an object. Possible alternative definitions of the FieldModifier declaration are the <javaStaticField>, <javaFinalField>, and <javaVolatileField> elements. The <javaSynchronizedMethod> element has the following predefined attributes:name ="transient" and value ="transient".
