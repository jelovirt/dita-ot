javaExceptionClassRef

This element provides a Java exception class topic type as a link in a DITA map.

Description

A <javaExceptionClassRef > can contain <topicref> and <topicmeta> elements, allowing you to express navigation or table-of-contents hierarchies, as well as implying relationships between the containing topicref and its children.

Example

<javaPackageRef navtitle="Package com.ibm.xtools.pluglets" id="com.ibm.xtools.patterns.pluglet"
        href="com.ibm.xtools.pluglets.dita" locktitle="yes">
     <javaClassRef navtitle="PlugletCompiler Class" href="PlugletCompiler.dita">
     </javaClassRef>
     <javaExceptionClassRef navtitle="PlugletException Class" href="PlugletException.dita">
     </javaExceptionClassRef>
</javaPackageRef>

Contained by

Contains

topicmeta, topicref

Content model

( (topicmeta) (optional) then (topicref) (any number))
