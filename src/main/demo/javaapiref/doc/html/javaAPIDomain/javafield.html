javafield

The <javafield> element is a reference to a Java field.

Description

The <javafield> element can be used to reference a Java field from another topic type.

Example

<task id="httpxml">
  <title>Getting an XML file through an HTTP connection</title>
  <taskbody>
    <steps>
    <step><cm>Open the connection</cm>
      <info>You can use <javafield keyref="com.ibm.xtools.patterns.framework.uml2.AbstractImportEngineTask.IDD">IDD field (AbstractImportEngineTask)</javafield> 
      in a Java application.</info>
    </step>
    ...
    </steps>
  </taskbody>
</task>
