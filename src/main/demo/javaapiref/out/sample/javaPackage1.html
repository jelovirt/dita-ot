Package org.topic.TopicCollection

This package supports managing collections of topics.

Interface summary
NodeAvailabilityThis interface represents the access to a node that may not be available.
NodeCompletenessThis interface represents the completeness of a node that may not be complete.
Class summary
TopicNodeThis class represents the position of a topic within a collection.
TreeNodeThis class represents the position of a topic within a tree.
Exception summary
CollectionTraversalExceptionThis class identifies an exception during a traverse operation on a collection.

Package org.topic.TopicCollection Description

The same topic can reside in many different collections.
