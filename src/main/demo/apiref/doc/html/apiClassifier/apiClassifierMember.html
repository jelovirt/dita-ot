apiClassifierMember

The <apiClassifierMember> element specifies that the classifier member is a substructure field of the classifier.

Description

The <apiClassifierMember> element is an object that is contained in a set (or more) of classes. A class can be derived from more than one base class. In a multiple-inheritance model (where classes are derived from more than one base class), the base classes are specified using the <apiClassifierMember> element.
