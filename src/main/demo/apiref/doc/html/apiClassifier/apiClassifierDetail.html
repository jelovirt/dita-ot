apiClassifierDetail

The <apiClassifierDetail> element provides detailed information about the structure and behavior of the classifier.

Description

In general, for expressing the detailed classifier in the HTML format, there are two exclusive alternatives, <apiSyntax> which supplies the syntax and <apiClassifierDef> to write custom processing for formatting. The <example> and <section> elements are provided within the <apiClassifierDetail> element to support the current API reference.
