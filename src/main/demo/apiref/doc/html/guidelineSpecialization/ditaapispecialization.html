DITA API specialization

The DITA API specialization documentation describes the Darwin Information Typing Architecture (DITA) specialization for application programming interfaces (APIs) topic types.

The documentation describes the XML elements that can be used to document generic and Java API libraries using the DITA API topic types.

If you have any questions related to the content, contact Mariana Alupului (malupulu@us.ibm.com).

  • Overview
    You can use the DITA API specialization to author and generate API reference documentation for Java and other programming languages.
  • Generic API topic types
    The generic API topic types describe any application programming interface (API) library and can serve as a basis for deriving more specialized topic types for specific languages or other library environments.
