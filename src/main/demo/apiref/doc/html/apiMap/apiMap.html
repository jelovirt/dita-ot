apiMap

The <apiMap> element provides the name of the application programming interface (API) reference and all the elements necessary to describe the package, or set of packages, and its components.

Description

The containing element for a map is the <apiMap> element that can contain the title and id attributes. In the <apiMap> element, use the <apiPackageRef>, <topicmeta> and <reltable> sub elements to add and organize references to the topics.

Contains

apiItemRef, reltable, topicmeta

Content model

( (topicmeta) (optional) then (apiItemRef) (any number) then (reltable) (any number))

Inheritance

map/map

  • apiItemRef
    This element provides a application programming interface (API) class topic type as a link in a DITA map.
