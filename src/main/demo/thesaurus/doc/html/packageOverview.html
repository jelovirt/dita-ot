Taxonomy specialization

The Taxonomy specialization supports semantic classification of content.

Description

You take the following approach:

  1. Define the subject matter for your content by writing subject topics. In effect, these topics define a single sense of a term from a glossary.
  2. Define relationships between these subjects with a scheme map.
  3. Classify your content by specifying relationships between content topics and these subject topics in your navigation map.
Note: This specialization conforms closely to the SKOS standard, which (though largely stable) hasn't progressed to full approval yet. Because of that, it is possible that this specialization could change in the future if SKOS changes during approval. The RDF output generated from this specialization is more likely to change than the markup itself. For more information about SKOS, see http://www.w3.org/2004/02/skos/core/.
