subjectNote

A miscellaneous public or private note about the subject definition.

Description

Use the subjectNote element only if none of the other subject definition notes has the right connotations. You can also specialize new kinds of notes about subjects from the subjectNote element. See also http://www.w3.org/TR/swbp-skos-core-guide/#secdocumentation.
