hasNarrower

Specifies that the contained hierarchy indicates a general-purpose, NARROWER-THAN relationship between subjects.

Description

Within the hierarchy, the child subject is narrower than the parent subject, but the way in which it is narrower isn't specified. For example, planting roses is a narrower subject than horticulture. See also http://www.w3.org/TR/swbp-skos-core-guide/#sechierarchy.
