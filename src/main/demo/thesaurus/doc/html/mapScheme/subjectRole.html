subjectRole

Specifies a role for the contained subjects.

Description

Within the subjectRelHeader element, the subjectRole element defines the role by reference to a subject.

Within the subjectRel element, the subjectRole element identifies the role played by a subject within an associative relationship.

subjectRel, subjectRelHeader
