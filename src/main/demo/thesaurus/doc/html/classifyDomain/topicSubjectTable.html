topicSubjectTable

Specifies a structured classification of content topics.

Description

The topicSubjectTable element provides a tabular structure for classifying content as an alternative to nesting the topicsubject element within a reference to the classified topic. Each row in the table identifies one or more topics and specifies the subjects that classify those topics. The columns of the table distinguish different kinds of subjects. In particular, the topic subject table can be useful for facet classification.

Contains

topicmeta, topicSubjectHeader, topicSubjectRow

Content model

( (topicmeta) (optional) then (topicSubjectHeader) (optional) then (topicSubjectRow) (one or more))

Inheritance
