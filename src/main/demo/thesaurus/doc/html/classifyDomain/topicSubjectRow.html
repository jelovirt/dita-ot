topicSubjectRow

Classifies one or more topics with a structured set of subjects.

Description

The first cell within the row contains one or more content topics. The remaining cells specify the subjects that classify the content topics. These subjects must conform to the constraints for their column as specified by the topicSubjectHeader element.
