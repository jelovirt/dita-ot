mymusic/ directory information

The files in the mymusic directory are used to generate topics using the songCollection specialization based on an iTunes library.

To generate your own online library, first export the library from iTunes into the mymusic directory. Keep the default name of Library.xml. Then, assuming you have the "full easy install" version of the toolkit, simply run the music.xml file (or run the "run.bat" batch file that calls the ant build). The library will be generated in the music/ subdirectory; open index.html to view your files.

This directory also contains the following important files:
  • itunexslSimple\simplifyLibrary.xsl creates a simplified version of the iTunes XSL that is easier to process in later steps.
  • itunexslSimple\customize.xsl allows you to specify playlists that you want to show up in your library; it also lets you specify artists, albums, or songs that you might not want to show off to your friends.
  • music\index.html sets up the frames for rendering your collection
  • music\musicbanner.html is a banner across the top of your collection that lets you select an alternate view
  • music\musicmap2bandlist.xsl is used by the left-hand navigation; it renders the ditamap as XHTML.
  • music\overview.html is the initial content frame displayed when you view the collection.
  • music\starter.html is the initial navigation frame on the left.
  • music\view.xsl renders the topics in the main content frame.
