cxxClassDerivations

The <cxxClassDerivations> element specifies a list of class derivations for a class.

Description

Used within a <cxxClassDefinition> element to specify any class derivations (base classes etc.) for a class. Must contain at least one <cxxClassDerivation> or <cxxStructDerivation> element.

Example

<cxxClassDefinition>
	<cxxClassDerivations>
		<cxxClassDerivation>
			...
		</cxxClassDerivation>
		...
	</cxxClassDerivations>
	<cxxClassAPIItemLocation>
		...
	</cxxClassAPIItemLocation>
<cxxClassDefinition>
