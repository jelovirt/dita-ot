cxxClassTemplateParameter

The <cxxClassTemplateParameter> element describes a class parameter.

Description

One or more <cxxClassTemplateParameter> elements reside within the <cxxClassTemplateParameters> container. <cxxClassTemplateParameter> requires the child element <cxxClassTemplateParameterType>; that describes the type of parameter. <cxxClassTemplateParameter> may also contain the child element <apiDefNote> to describe the parameter further.

Contained by

cxxClassTemplateParameters

Contains

apiDefNote, cxxClassTemplateParameterType
