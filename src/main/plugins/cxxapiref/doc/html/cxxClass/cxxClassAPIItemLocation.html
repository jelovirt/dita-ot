cxxClassAPIItemLocation

The <cxxClassAPIItemLocation> element is defined within <cxxClassDefinition>. <cxxClassAPIItemLocation> specifies the location of the API.

Description

The <cxxClassAPIItemLocation> element must contain the location of the API declaration using <cxxClassDeclarationFile> and <cxxClassDeclarationFileLine>; the location of the API definition is optional.

Example

<cxxClassAPIItemLocation>
				<cxxClassDeclarationFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxClassDeclarationFileLine name="lineNumber" value="775"/>
				<cxxClassDefinitionFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxClassDefinitionFileLineStart name="lineNumber" value="765"/>
				<cxxClassDefinitionFileLineEnd name="lineNumber" value="787"/>
			</cxxClassAPIItemLocation>
