cxxClassTemplateParameters

The <cxxClassTemplateParameters> element contains a set of class parameters.

Description

The <cxxClassTemplateParameters> element contains one or more <cxxClassTemplateParameter> elements. Each of which describes a class parameter. <cxxClassTemplateParameters> is declared within <cxxClassDefinition>

Example

<cxxClassTemplateParameters>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename T</cxxClassTemplateParameterType>
    <apiDefNote>The type of the first function object. </apiDefNote>
  </cxxClassTemplateParameter>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename U</cxxClassTemplateParameterType>
    <apiDefNote>The type of the second function object. </apiDefNote>
  </cxxClassTemplateParameter>
</cxxClassTemplateParameters>
