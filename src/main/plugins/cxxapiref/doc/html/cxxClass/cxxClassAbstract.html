cxxClassAbstract

The <cxxClassAbstract> element contains optional abstract information.

Description

The <cxxClassAbstract> element resides within the <cxxClassDefinition> element. The cxxClassAbstract element has no children but has the attributes name and value. name and value are both set to "abstract".
