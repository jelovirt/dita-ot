cxxDefineDetail

The <cxxDefineDetail> element provides detailed information about the structure and behaviors of the C++ define.

Description

The <cxxDefineDetail> element is defined within <cxxDefine>. In <cxxDefineDetail> the <example> and <section> elements are provided to support the current specific C++ API reference.

cxxDefine
