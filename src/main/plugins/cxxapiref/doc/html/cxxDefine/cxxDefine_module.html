cxxDefine module

Defines a top-level container for all the elements necessary to describe a define.

Description

The <cxxDefine> topic type contains elements necessary to show the name of the define, the define syntax, and the usage description of the define. This topic type module also contains all the elements necessary to describe the component variables and functions.

The cxxDefine topic contains the following elements:

  • cxxDefine
    The <cxxDefine> element provides the name of the C++ API define, all the elements necessary to describe the members that are introduced by the define declaration.
  • cxxDefineAccessSpecifier
    The <cxxDefineAccessSpecifier> element describes the level of access the define allows.
  • cxxDefineAPIItemLocation
    The <cxxDefineAPIItemLocation> element is defined within <cxxDefineDefinition>. <cxxDefineAPIItemLocation> specifies the location of the API.
  • cxxDefineDeclarationFile
    The <cxxDefineDeclarationFile> element is defined within <cxxDefineAPIItemLocation>. <cxxDefineDeclarationFile> specifies the file in which the API declaration is present.
  • cxxDefineDeclarationFileLine
    The <cxxDefineDeclarationFileLine> element is defined within <cxxDefineAPIItemLocation>. <cxxDefineDeclarationFileLine> specifies the line on which the API is declared
  • cxxDefineDefinition
    The <cxxDefineDefinition> element is defined within <cxxDefineDetail>. <cxxDefineDefinition> represents the C++ define declaration.
  • cxxDefineDetail
    The <cxxDefineDetail> element provides detailed information about the structure and behaviors of the C++ define.
  • cxxDefineNameLookup
    Provides a unique identifier of the define.
  • cxxDefineParameter
    The <cxxDefineParameter> element describes a define parameter.
  • cxxDefineParameterDeclarationName
    The <cxxDefineParameterDeclarationName> element specifies the name of the parameter used in the declaration of the define.
  • cxxDefineParameters
    Element that contains information about the define identifier-list.
  • cxxDefinePrototype
    The <cxxDefinePrototype> element specifies the define declaration.
  • cxxDefineReimplemented
    The <cxxDefineReimplemented> element identifies defines that have been re-implemented.
