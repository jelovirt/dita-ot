cxxDefineParameter

The <cxxDefineParameter> element describes a define parameter.

Description

The <cxxDefineParameter> element describes a single parameter in an identifier-list.

Example

<cxxDefineDefinition>
    <cxxDefineAccessSpecifier value="public"/>
    <cxxDefinePrototype>#define MergeModes(a, b)</cxxDefinePrototype>
    <cxxDefineNameLookup>MergeModes</cxxDefineNameLookup>
    <cxxDefineParameters>
        <cxxDefineParameter>
            <cxxDefineParameterDeclarationName>a</cxxDefineParameterDeclarationName>
            <apiDefNote/>
        </cxxDefineParameter>
        <cxxDefineParameter>
            <cxxDefineParameterDeclarationName>b</cxxDefineParameterDeclarationName>
            <apiDefNote/>
        </cxxDefineParameter>
    </cxxDefineParameters>
    <cxxDefineAPIItemLocation>
        <cxxDefineDeclarationFile name="filePath" value="C:/EPOC/master/sf/os/commsfw/serialserver/c32serialserver/SCOMM/CS_PORT.CPP"/>
        <cxxDefineDeclarationFileLine name="lineNumber" value="126"/>
    </cxxDefineAPIItemLocation>
</cxxDefineDefinition>
