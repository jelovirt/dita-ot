cxxDefineReimplemented

The <cxxDefineReimplemented> element identifies defines that have been re-implemented.

Description

This identifies and links to defines that have been re-implemented.

<cxxDefineReimplemented> has no compulsatory attributes, but can contain the attributes keyref, type, format and scope. scope can be set to one of three values, "local", "peer" or "external".

Example

<cxxDefineReimplemented href="class_c_active.xml#class_c_active_1a4bb7613b">CActive::iSpare</cxxDefineReimplemented>
