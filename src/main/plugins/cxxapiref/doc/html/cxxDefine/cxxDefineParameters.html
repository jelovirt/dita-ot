cxxDefineParameters

Element that contains information about the define identifier-list.

Description

This element contains information about the define identifier-list that identifies any parameters in the function-like macro.

Example

<cxxDefineParameters>
    <cxxDefineParameter>
        <cxxDefineParameterDeclarationName>prmIndex</cxxDefineParameterDeclarationName>
        <apiDefNote/>
    </cxxDefineParameter>
</cxxDefineParameters>
