cxxDefineDeclarationFile

The <cxxDefineDeclarationFile> element is defined within <cxxDefineAPIItemLocation>. <cxxDefineDeclarationFile> specifies the file in which the API declaration is present.

Description

The element <cxxDefineDeclarationFile> is empty but for the required attributes name and value. name is always set to "filePath" and value is set to the file path.

Example

<cxxDefineDeclarationFile name="filePath" value="C:/EPOC/master/sf/os/commsfw/commsfw/src/cfperfmetrics.cpp"/>
