cxxUnionAbstract

The <cxxUnionAbstract> element contains optional abstract information.

Description

The <cxxUnionAbstract> element resides within the <cxxUnionDefinition> element. The cxxUnionAbstract element has no children but has the attributes name and value. name and value are both set to "abstract".
