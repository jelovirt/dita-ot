cxxUnionDeclarationFileLine

The <cxxUnionDeclarationFileLine> element is defined within <cxxUnionAPIItemLocation>. <cxxUnionDeclarationFileLine> specifies the line on which the API is declared

Description

The element <cxxUnionDeclarationFileLine> is empty but for the required attributes name and value. Name is always lineNumber

Example

<cxxUnionDeclarationFileLine name="lineNumber" value="59"/>
