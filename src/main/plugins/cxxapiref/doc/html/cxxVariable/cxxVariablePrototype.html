cxxVariablePrototype

The <cxxVariablePrototype> element provides a representation of the variable declaration.

Description

The optional <cxxVariablePrototype> element has no children and does not require any default attributes.

Example

<cxxVariablePrototype>TRequestStatus iStatus</cxxVariablePrototype>
