cxxVariableDetail

The <cxxVariableDetail> element provides detailed information about the structure and behaviors of a C++ variable.

Description

The <cxxVariableDetail> element is defined within <cxxVariable>. In <cxxVariableDetail> the <example> and <section> elements are provided to support the current specific C++ API reference.

cxxVariable
