cxxVariableStorageClassSpecifierStatic

The <cxxVariableStorageClassSpecifierStatic> element marks this variable as static.

Description

The <cxxVariableStorageClassSpecifierStatic> element has no children. <cxxVariableStorageClassSpecifierStatic> can have the fixed attributes name and value both of which are set to "static".

Example

<cxxVariableDefinition>
    <cxxVariableAccessSpecifier value="private"/>
    <cxxVariableStorageClassSpecifierStatic/>
    <cxxVariableDeclaredType>const <apiRelation href="e32def_8h.xml#e32def_8h_1a8094a5c8ef3bcf6221e5ff6405127d34">TText</apiRelation> *const</cxxVariableDeclaredType>
    <cxxVariableScopedName>GlyphSample</cxxVariableScopedName>
    <cxxVariablePrototype>static const TText *const KTScript2GlyphSample[]</cxxVariablePrototype>
    <cxxVariableNameLookup>GlyphSample::KTScript2GlyphSample</cxxVariableNameLookup>
    <cxxVariableAPIItemLocation>
        <cxxVariableDeclarationFile name="filePath" value="C:/epoc32/include/graphics/gdi/glyphsample.h"/>
        <cxxVariableDeclarationFileLine name="lineNumber" value="53"/>
    </cxxVariableAPIItemLocation>
</cxxVariableDefinition>
