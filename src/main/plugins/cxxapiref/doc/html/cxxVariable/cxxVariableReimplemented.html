cxxVariableReimplemented

The <cxxVariableReimplemented> element identifies variables that have been re-implemented.

Description

This identifies and links to variables that have been re-implemented.

<cxxVariableReimplemented> has no compulsatory attributes.

Example

<cxxVariableReimplemented href="class_c_active.xml#class_c_active_1a4bb76130b67b5430b3b736194df7edeb">CActive::iSpare</cxxVariableReimplemented>
