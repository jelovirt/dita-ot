cxxVariableScopedName

The <cxxVariableScopedName> element holds the scope of the variable i.e. the declaration of the enclosing namespace/class/struct/union.

Description

The <cxxVariableScopedName> has no compulsory attributes and no child elements.

Example

<cxxVariableScopedName>CServer2</cxxVariableScopedName>
