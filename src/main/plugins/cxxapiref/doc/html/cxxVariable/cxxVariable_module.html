cxxVariable module

Defines a top-level container for all the elements necessary to describe a variable.

Description

The <cxxVariable> topic type contains elements necessary to show the name of the variable, a short description and the variable definition.

Common elements used to describe a variable are:

  • cxxVariableAccessSpecifier
  • cxxVariableDeclaredType
  • cxxVariableScopedName
  • cxxVariablePrototype
  • cxxVariableNameLookup
  • cxxVariableAPIItemLocation

The cxxVariable topic contains the following elements:

  • cxxVariable
    The <cxxVariable> element contains the elements necessary to describe a variable.
  • cxxVariableAccessSpecifier
    The <cxxVariableAccessSpecifier> element specifies the level of access the variable has.
  • cxxVariableAPIItemLocation
    The <cxxVariableAPIItemLocation> element specifies the location of the API.
  • cxxVariableConst
    The <cxxVariableConst> element contains optional const information.
  • cxxVariableDeclarationFile
    The <cxxVariableDeclarationFile> element is defined within <cxxVariableAPIItemLocation>. <cxxVariableDeclarationFile> specifies the file in which the API declaration is present.
  • cxxVariableDeclarationFileLine
    The <cxxVariableDeclarationFileLine> element is defined within <cxxVariableAPIItemLocation>. <cxxVariableDeclarationFileLine> specifies the line on which the API is declared.
  • cxxVariableDeclaredType
    The <cxxVariableDeclaredType> element specifies the variable data type in the declaration.
  • cxxVariableDefinition
    The <cxxVariableDefinition> element specifies the variable definition.
  • cxxVariableDetail
    The <cxxVariableDetail> element provides detailed information about the structure and behaviors of a C++ variable.
  • cxxVariableNameLookup
    The <cxxVariableNameLookup> element specifies the lookup name of a variable.
  • cxxVariablePrototype
    The <cxxVariablePrototype> element provides a representation of the variable declaration.
  • cxxVariableReimplemented
    The <cxxVariableReimplemented> element identifies variables that have been re-implemented.
  • cxxVariableScopedName
    The <cxxVariableScopedName> element holds the scope of the variable i.e. the declaration of the enclosing namespace/class/struct/union.
  • cxxVariableStorageClassSpecifierExtern
    The <cxxVariableStorageClassSpecifierExtern> element marks this variable as extern.
  • cxxVariableStorageClassSpecifierMutable
    The <cxxVariableStorageClassSpecifierMutable> element marks this variable as mutable.
  • cxxVariableStorageClassSpecifierStatic
    The <cxxVariableStorageClassSpecifierStatic> element marks this variable as static.
  • cxxVariableVolatile
    The <cxxVariableVolatile> element marks this variable as volatile.
